Skolprojekt och kurser:

Frontend 1 - JavaScript

Grundläggande programmering. Utveckla programmeringslösningar i JavaScript. Förståelse för yrkesrollen. Relationen mellan frontend, backend, och fullstack. Praktiskt har jag fått öva på att göra enklare applikationer i JavaScript samt skriva, testa och felsöka kod. Manipulera DOM med hjälp av JavaScript och HTML. Grundläggande versionshantering med GIT. Fetch och REST. Den individuella uppgiften så valde jag att göra ett program tänk att användas som ett hjälpmedel vid marknadsundersökningar. Uppgiften handlade om att sortera information på olika sätt. Fokuset låg på programmets funktionalitet, den tekniska dokumentationen och inte det estetiska i denna kurs.

Affärsmannaskap

En kurs där syftet var att få djupare förståelse för hur man kan bidra till varje verksamhets resultat och skapa lönsamhet både på kort och lång sikt. Identifiera nuvarande och framtida kompetensbehov inom företag och branschen.

Frontend 2 - HTML och CSS

Denna kurs gav mig kunskaper inom HTML och CSS för att kunna skapa en interaktiv upplevelse med ett dynamiskt innehåll på en webbplats. Skapa applikationer med HTML, CSS och JavaScript. Olika layoutmodeller som används för utveckling av webbsidor. SASS/LESS och hur det fungerar. Koppling mellan DOM och HTML. Skapa responsiva webblösningar. Vikten av användbarhet, olika tillgänglighetsstandarder. Agil projektmetodik. Har under denna kurs även använt Bootstrap för att skapa webbapplikationer. SCSS, jQuery. Gruppuppgift, bygga en sida som liknar Twitters hemskärm vid inloggat läge med hjälp av Bootstrap och CSS. Individuell Uppgift. SCSS. En landing-page. E-handelssida, frontpage, produktsida, lägg-till-produkt-sida, Om oss-sida. Animationer och rörliga element samt responsiv på mobila enheter.

Frontend 3 - JavaScript-bibliotek och ramverk

SPA- single page application. Anropa API:er. React. Individuell uppgift Sportify. En single page application gjord med React Material UI. SASS.

Databasteknik

Registrering och login gjord med react och MySQL Relationsdatabasen MySQL. Gruppuppgift: en databas för en skola med lärare, elever, kurser och betyg. Individuell uppgift: en databas för biograf med tre tabeller: film, föreställning och biograf.

Backend 1
