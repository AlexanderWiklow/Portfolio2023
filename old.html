Backend

Databaser spelar en avgörande roll i modern webbutveckling och låter oss lagra och hämta stora mängder information. För mitt skolprojekt har jag valt att använda MySQL som mitt databashanteringssystem och har kombinerat det med kraften i Express och Node.js för att skapa en dynamisk webbapplikation. Projektet är ett biobokningssystem som kräver en databas för att lagra information om filmer, speltider, biljetter, biografer, städer samt föreställningar. Genom att använda MySQL kan jag enkelt hantera denna data och se till att den är säker och organiserad. En annan applikation hanterar inloggnings- och registreringssida för applikationen. Denna funktion krävde en robust databas för att lagra användarinformation och hålla reda på deras transaktioner. Sammanfattningsvis har kombinationen av MySQL, Express och Node.js varit avgörande för att utveckla en funktionell och pålitlig databas för mitt skolprojekt.

Frontend

Efter mina frontend-studier har jag skaffat mig erfarenhet av att skapa engagerande och responsiva användargränssnitt för webbapplikationer. Mina studier har inkluderat HTML, SCSS, React, Material UI, JSON Server, JavaScript, och Bootstrap som alla är viktiga teknologier för att bygga moderna front-end-applikationer. Ett av mina projekt var att skapa en landing page för en fiktiv e-handelssida som heter "Sportify". Jag använde HTML, CSS och React för att designa och implementera sidans layout och funktionalitet, samtidigt som jag använde Material UI för att säkerställa att den var responsiv och visuellt tilltalande. Ett annat projekt var skapandet av en landningssida för ett företag för friluftsutrustning som heter "Hiking & Stuff." För det här projektet använde jag JavaScript och SCSS för att bygga ett dynamiskt och interaktivt användargränssnitt, och jag använde JSON Server för att använda ett backend-API. Dessa projekt har gjort det möjligt för mig att få praktisk erfarenhet av att använda de teknologier jag har studerat, och jag är säker på min förmåga att skapa högkvalitativa front-end-gränssnitt för alla webbapplikationer.
